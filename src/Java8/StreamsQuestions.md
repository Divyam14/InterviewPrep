# Coding Questions with Increasing Complexity
    
## Level 1 – Basic Transformations

Given a list of integers, return squares.

Filter odd numbers.

Convert list of names to uppercase.

Count items starting with a letter.

Given a list of strings, find the longest string.

## Level 2 – Combined Operations

Remove duplicates and sort numbers.

Given Employee objects, collect names.

Check if any employee has salary > X.

Find first element matching a condition.

Generate first 10 even numbers using IntStream.

## Level 3 – FlatMap Usage

Flatten List<List<Integer>> into List<Integer>.

Flatten List<String> where each string is comma-separated.

Given Students with List<Subjects>, list all subjects.

## Level 4 – Grouping and Aggregation

Group employees by department.

Average salary by department.

Count frequency of words in text.

Find top 3 salaried employees.

## Level 5 – Reduction + Custom Comparators

Sum all transaction amounts via reduce.

Find max salary via reduce.

Implement a custom collector to compute summary stats.

## Level 6 – Numeric Streams and Performance

Use IntStream to compute squares sum of 1–100.

Compare performance of boxed Stream<Integer> vs IntStream.

## Level 7 – Short-circuit Logic

Given huge list, detect if any value > threshold.

Return first name ending with "son".

## Level 8 – Infinite Streams and Complex Pipelines

Generate Fibonacci sequence using iterate().

Generate random numbers until > 0.95.

Produce primes using streams (advanced).

## Level 9 – Parallel Streams Concepts

Parallel sort vs sequential.

Parallel reduce hazards with mutable accumulators.

Performance measurement for >100K inputs.

## Level 10 – Hard Interview Problems

Word frequency sorted by highest frequency.

Transaction analytics (filter, groupBy, sum).

Longest increasing subsequence using Streams.

Paginate list using skip + limit.

Sliding window using Streams (hard).
